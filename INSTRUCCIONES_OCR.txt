VIDPASE + FIREBASE

1. Descomprime este ZIP.
2. Entra al repositorio: vidpase/vidpase.github.io
3. Pulsa Add file > Upload files.
4. Sube index.html y app.js en la raiz del repositorio.
5. Confirma el reemplazo de index.html y pulsa Commit changes.
6. Espera 1-3 minutos y abre https://vidpase.github.io/
7. Presiona Ctrl + F5.

El panel ahora pide el correo y contraseña del administrador de Firebase.

Incluye:
- Firebase Authentication para el administrador.
- Inicio anonimo para visitantes.
- Firestore para campañas, estadísticas y registros.
- Storage para portadas, miniaturas y capturas.
- Panel de capturas.
- Migración de campañas que estaban en localStorage.
- Enlace de YouTube opcional y miniatura automática.
- Miniatura manual opcional.
- Banco de hasta 50 ideas, mostrando 3 al azar.
- Campo HTML para widget publicitario y vista previa.

IMPORTANTE:
El enlace final se guarda temporalmente dentro del documento público de la campaña para que el flujo funcione ahora. En la siguiente etapa debe entregarse mediante Cloud Functions para evitar que un usuario avanzado pueda encontrarlo en Firestore.
